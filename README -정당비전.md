# 🏛️ AI 정당 운영 시스템
## 1인 AI 정당 자율 운영 플랫폼

**목표**: 2028년 4월 국회의원 선거에서 10석 당선 (지역구 3명 + 비례대표 7명)

---

## 🌊 당의 비전 — 말씀이 흐르는 사회

> "말씀이 개인에게 흐르면 신앙 인격이 되고, 말씀이 사회에 흐르면 공동체 헌법이 된다."

이 정당은 단순한 기술 정당이 아닙니다.
인류 역사상 구현하지 못한 **말씀이 통치하는 공동체**를 대한민국과 통일한국에 세우기 위한 정치적 플랫폼입니다.

### 두 개의 흐름

**흐름 1 — 개인**: 말씀이 한 사람에게 흐를 때 신앙 인격이 형성됩니다.
이것은 교회와 가정이 담당합니다.

**흐름 2 — 사회**: 말씀이 공동체 시스템에 흐를 때 하나님 나라의 모형이 됩니다.
이것은 헌법, 법률, 온톨로지 플랫폼이 담당합니다. **이것이 이 정당의 사명입니다.**

---

## 🧬 당의 철학 — 온톨로지 공화국

미국이 청교도 신앙 위에 자유와 평등의 공화국을 세웠다면,
우리는 하나님 말씀 위에 **은사와 필요가 연결되는 공동체 공화국**을 세웁니다.

### 온톨로지 7대 노드

```
           [말씀]
          /  |  \
      [사람][은사][지역]
          \  |  /
       [필요][자원]
             |
          [시스템]
```

| 노드 | 정치적 의미 |
|------|-----------|
| 말씀 | 헌법의 정신적 기반 |
| 사람 | 주권자 시민 |
| 은사 | 개인의 재능과 직업 |
| 필요 | 사회적 약자와 공공재 |
| 자원 | 국가 예산과 민간 자원 |
| 지역 | 지방 자치와 풀뿌리 민주주의 |
| 시스템 | AI 기반 정책 실행 플랫폼 |

### 4대 시스템 공약

1. **은사 등록 시스템** — 모든 시민의 재능이 국가에 등록되고 연결되는 사회
2. **필요 매칭 시스템** — 사회적 약자의 필요가 자동으로 연결되는 사회
3. **지역 커뮤니티 시스템** — 마을 단위 자치 공동체가 살아있는 사회
4. **자동 매칭 플랫폼** — AI가 은사와 필요를 연결하여 "이웃 사랑"이 시스템으로 작동하는 사회

### 통일한국 비전

이 시스템의 최종 목적지는 **통일 후 북한 180개 지역**에 온톨로지 공동체를 세우는 것입니다.
대한민국에서 파일럿 검증 → 통일한국 전면 적용.

## 🎯 프로젝트 개요

이 시스템은 1명의 대표가 AI 에이전트 팀을 활용하여 정당을 운영하는 혁신적인 플랫폼입니다.

### 핵심 철학
- **말씀 통치**: 하나님 말씀이 정책 판단의 최종 기준
- **투명성**: 모든 의사결정 과정 공개
- **참여**: 시민이 직접 정책 제안 및 투표
- **효율성**: AI 자동화로 24/7 운영
- **책임성**: 인간이 최종 결정, AI는 보조
- **공동체성**: 개인의 은사가 공동체의 필요와 연결되는 사회

---

## 🤖 AI 에이전트 팀

1. **중앙 오케스트레이터** - 전체 시스템 조정
2. **고객지원팀** - 24/7 챗봇 상담
3. **마케팅팀** - SNS 콘텐츠 자동 생성
4. **데이터분석팀** - 여론 분석 및 예측
5. **정책개발팀** - 시민 제안 분석 및 정책화
6. **의정활동감시팀** - 국회의원 활동 모니터링
7. **기술개발팀** - 플랫폼 유지보수
8. **후보자관리팀** - 인재 발굴 및 교육

## 🚀 빠른 시작

### 1. 환경 설정
```bash
cd ai-political-party
pip install -r requirements.txt
cp config/.env.example config/.env
# .env 파일에 API 키 입력
```

### 2. 시스템 실행
```bash
# 오케스트레이터 실행 (백그라운드)
python agents/orchestrator.py --mode auto &

# 대시보드 실행
streamlit run dashboard/control_panel.py
```

### 3. 웹사이트 실행
```bash
cd web/frontend
npm install
npm run dev
```
