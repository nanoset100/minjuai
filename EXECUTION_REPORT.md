## 🚀 즉시 실행 가능

### 실행 방법

```bash
cd ai-political-party

# 1. 환경 변수 설정
cp config/.env.example config/.env
# .env 파일에 ANTHROPIC_API_KEY 입력

# 2. 시스템 실행
./start.sh
```

### 테스트 시나리오

**시나리오 1: 오케스트레이터 테스트**
```bash
python agents/orchestrator.py
```
→ 일일 브리핑 자동 생성됨!

**시나리오 2: 챗봇 테스트**

→ 4가지 질문에 자동 응답!

**시나리오 3: 웹사이트 체험**
```bash
./start.sh

# 브라우저에서 web/frontend/index.html 열기
```
