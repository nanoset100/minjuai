# 민주AI 핵심 기능 6가지 상세 설명

> 세계 최초 AI 운영 정당 플랫폼 — MinjuAI
> 데이터 기반, 시민 참여, 완전 투명한 정치의 새로운 패러다임

## 1. 주간 정책 연구 (Weekly Policy Research)

### 개요
매주 1개 핵심 주제를 AI가 자동 선정하여 심층 연구하는 기능입니다.
1년 52주 = 52개 정책 주제를 체계적으로 다루는 **정책 온톨로지(Policy Ontology)** 시스템입니다.

### 작동 방식
1. **주제 자동 선정**: 경제 → 교육 → 환경 → 복지 → 외교 → 기술 순환
2. **데이터 수집**: 통계청, OECD, 학술논문, 해외 사례 자동 수집
3. **글로벌 비교**: 한국 vs 미국 vs 일본 vs EU 등 국제 비교 분석
4. **정책안 도출**: 연구 결과를 바탕으로 실현 가능한 정책안 자동 생성
5. **시민 공유**: 연구 결과를 앱과 SNS에 자동 게시

### 예시
```
제12주 주제: 청년 주거 정책
├── 한국 현황: 청년 주거비 부담률 42% (OECD 평균 28%)
├── 해외 사례: 싱가포르 HDB, 오스트리아 빈 사회주택
├── 정책안: 청년 공공임대 30만호 + 주거수당 월 20만원
└── 예산 분석: 연간 3.2조원 필요, 재원 조달 방안 3가지
```

### 기존 정당과의 차이
| 기존 정당 | 민주AI |
|-----------|--------|
| 선거 때만 정책 발표 | 매주 꾸준히 연구 |
| 소수 전문가가 작성 | AI + 데이터 기반 |
| 국내 사례 중심 | 글로벌 비교 분석 |
| 발표 후 검증 없음 | 실현가능성 AI 평가 포함 |

## 2. 솔루션 엔진 (Solution Engine)

### 개요
시민이 사회 문제를 질문하면, AI가 **근거 기반 해결책**을 제시하는 기능입니다.
단순 의견이 아닌, 데이터와 사례에 기반한 정책 솔루션을 제공합니다.

### 작동 방식
1. **질문 입력**: "청년 실업 문제 어떻게 해결할 수 있나요?"
2. **문제 분석**: AI가 질문을 분석하고 관련 데이터 수집
3. **해결책 도출**: 국내외 성공 사례 + 학술 연구 기반 솔루션 생성
4. **근거 제시**: 각 해결책에 대한 통계, 사례, 예산 근거 첨부
5. **실현가능성 평가**: 법적·행정적·재정적 실현 가능성 점수

### 예시
```
질문: "미세먼지 문제를 어떻게 해결할 수 있나요?"

솔루션 엔진 답변:
┌─────────────────────────────────────────────┐
│ 해결책 1: 석탄발전소 단계적 폐쇄 (실현가능성 85%)    │
│ - 근거: EU 사례, 독일 2030년 완전 탈석탄                │
│ - 예산: 5.2조원 (고용전환 비용 포함)                    │
│                                                         │
│ 해결책 2: 전기차 보급 가속 (실현가능성 78%)             │
│ - 근거: 노르웨이 전기차 점유율 80%                      │
│ - 예산: 연간 1.8조원 보조금                             │
│                                                         │
│ 해결책 3: 중국 협력 외교 강화 (실현가능성 62%)          │
│ - 근거: 한중 환경협력 MOU 기반                          │
└─────────────────────────────────────────────┘
```

### 핵심 가치
- **감정 아닌 데이터**: 이념이 아닌 근거에 기반한 답변
- **양비론 탈피**: 구체적 수치와 실행 방안 제시
- **시민 누구나**: 전문가가 아니어도 정책 솔루션에 접근 가능

## 3. 시민 참여 (Citizen Participation)

### 개요
시민이 직접 정책을 제안하고, AI가 이를 분석·평가하여 당 정책에 반영하는 기능입니다.
**진정한 직접 민주주의**를 기술로 구현합니다.

### 참여 방식
1. **정책 제안**: 시민이 자유롭게 정책 아이디어 제출
2. **AI 분석**: 실현가능성, 예산, 법적 타당성 자동 분석
3. **시민 투표**: 다른 시민들이 찬반 투표
4. **정책 채택**: 일정 지지율 이상이면 당 공식 정책으로 채택
5. **이행 추적**: 채택된 정책의 이행 과정 투명 공개

### 예시
```
시민 제안: "모든 공공건물 옥상에 태양광 패널 의무 설치"

AI 분석 결과:
├── 실현가능성: 72점/100점
├── 예산: 연간 8,200억원
├── 법적 검토: 건축법 시행령 개정 필요
├── 해외 사례: 프랑스 (2023년 의무화)
├── 예상 효과: 연간 120만 MWh 생산
└── 시민 투표: 찬성 78% | 반대 12% | 보류 10%

→ 결과: 당 정책으로 채택!
```

### 기존 정치와의 차이
| 기존 정치 | 민주AI |
|-----------|--------|
| 국민청원 → 답변만 | 제안 → AI분석 → 투표 → 채택 |
| 정치인이 정책 결정 | 시민이 직접 제안·투표 |
| 결과 불투명 | 전 과정 투명 공개 |
| 전문가만 참여 가능 | 누구나 쉽게 참여 |

## 4. 정책 온톨로지 (Policy Ontology)

### 개요
모든 정책을 **체계적인 지식 그래프(Knowledge Graph)**로 연결하는 시스템입니다.
"교육 정책이 경제에 미치는 영향", "환경 정책과 일자리의 관계" 같은 정책 간 연결을 AI가 자동 분석합니다.

### 구조
```
정책 온톨로지 구조 (52주 × 연간)

경제 ─┬── 고용 ─── 청년실업 ─── 교육과 연결
      ├── 부동산 ── 주거안정 ─── 복지와 연결
      └── 산업 ─── 기술혁신 ─── 환경과 연결

교육 ─┬── 초중고 ── 학교 교육 ── 예산과 연결
      ├── 대학 ─── 등록금 ──── 경제와 연결
      └── 평생 ─── 재교육 ──── 고용과 연결

환경 ─┬── 에너지 ── 탈원전 ──── 산업과 연결
      ├── 기후 ─── 탄소중립 ─── 외교와 연결
      └── 자원 ─── 순환경제 ─── 경제와 연결
```

### 핵심 가치
- **단편적 정책 탈피**: 하나의 정책이 다른 분야에 미치는 영향까지 분석
- **일관성 보장**: 경제 정책과 환경 정책이 모순되지 않도록 AI가 검증
- **축적 효과**: 매주 연구가 쌓이면서 점점 더 풍부한 지식 그래프 구축
- **글로벌 비교**: 각 주제별 세계 주요국 정책 비교 데이터 포함

## 5. 의정 활동 감시 (Legislative Monitoring)

### 개요
**국회 열린데이터 API**를 연동하여 현역 국회의원 295명의 의정활동을 실시간 모니터링합니다.
AI가 자동으로 활동이 저조한 의원과 취약 지역구를 발굴합니다.

### 데이터 출처
- **국회 열린데이터 포털** (open.assembly.go.kr) 공식 API
- 국회의원 인적사항 API
- 발의법률안 API
- 실시간 데이터 수집 및 분석

### 모니터링 항목
| 항목 | 설명 |
|------|------|
| 법안 발의 수 | 의원이 발의한/공동발의한 법안 수 |
| 소속 위원회 | 어떤 위원회에서 활동하는지 |
| 선수(재선 횟수) | 초선/재선/3선 등 경력 |
| 지역구/비례 | 지역구 의원인지 비례대표인지 |
| 경력 사항 | 의원의 전체 이력 |

### 현재 실제 데이터 (2026년 3월 기준)
```
총 모니터링: 22대 국회의원 295명
추적 법안:   25,337건
지역구 의원: 249명

취약 지역구 Top 5:
1. 서울 노원구갑 - 우원식 (무소속, 5선) - 발의 0건 [최우선]
2. 서울 영등포구을 - 김민석 (민주당, 4선) - 발의 3건 [우선]
3. 서울 동작구갑 - 김병기 (무소속, 3선) - 발의 3건 [우선]
4. 서울 동대문구갑 - 안규백 (민주당, 5선) - 발의 3건 [우선]
5. 경기 구리시 - 윤호중 (민주당, 5선) - 발의 3건 [우선]
```

### AI 분석 기능
- **의원별 성적표**: GPT-4o mini가 의원의 의정활동을 종합 평가
- **공략 전략 자동 생성**: 취약 지역구별 출마 전략 AI 생성
- **정당별/지역별 비교**: 어느 정당이, 어느 지역이 활동이 저조한지 분석

### 핵심 가치
- **실제 데이터**: 가상이 아닌 국회 공식 API의 실제 데이터
- **객관적 평가**: 이념이 아닌 수치 기반 의원 평가
- **시민의 알 권리**: 내 지역구 의원이 얼마나 일하는지 누구나 확인

## 6. 완전 투명성 (Full Transparency)

### 개요
AI 정당의 모든 의사결정 과정, 비용, 활동 내역을 **100% 공개**하는 기능입니다.
기존 정당의 불투명한 운영과 완전히 다른, **투명성 혁명**입니다.

### 투명 공개 항목

#### 1) AI 의사결정 과정
```
정책 결정 로그 (모두 공개):
├── 입력: 어떤 데이터를 AI에 넣었는지
├── 분석: AI가 어떤 논리로 판단했는지
├── 출력: 최종 결론은 무엇인지
└── 근거: 참고한 데이터/논문/사례 목록
```

#### 2) 비용 투명성
```
비용 대시보드 (/api/cost/stats):
├── AI API 사용 비용: 월별 상세 내역
├── 서버 운영 비용: Railway 호스팅 비용
├── 모델별 비용: GPT-4o mini 호출 횟수/비용
└── 절감 현황: 캐싱으로 절감된 비용
```

#### 3) 에이전트 활동 로그
```
실시간 활동 로그 (최근 100개 유지):
├── [21:30] 정책연구팀: 제12주 주거정책 연구 시작
├── [21:35] 의정감시팀: 국회 데이터 수집 완료 (295명)
├── [21:40] 마케팅팀: SNS 게시물 5개 생성
├── [21:45] 시민상담팀: 질문 23건 자동 응답
└── [21:50] 데이터분석팀: 주간 트렌드 리포트 생성
```

#### 4) 시민 참여 데이터
```
참여 통계 (실시간 공개):
├── 총 시민 제안: 1,247건
├── AI 분석 완료: 1,198건
├── 시민 투표 참여: 8,432명
├── 정책 채택: 23건
└── 이행 중: 15건
```

### 기존 정당과의 차이
| 기존 정당 | 민주AI |
|-----------|--------|
| 정치자금 불투명 | 모든 비용 실시간 공개 |
| 의사결정 과정 비공개 | AI 로그 전체 공개 |
| 활동 보고서 연 1회 | 실시간 활동 대시보드 |
| 당원만 정보 접근 | 누구나 열람 가능 |

## 요약: 6가지 핵심 기능의 시너지

```
주간 정책 연구 ──→ 정책 온톨로지에 축적
       ↓                    ↓
솔루션 엔진 ←── 축적된 지식으로 답변 품질 향상
       ↓
시민 참여 ──────→ 시민 제안이 다시 연구 주제로
       ↓
의정 활동 감시 ──→ 현역 의원 성적표로 차별화
       ↓
완전 투명성 ────→ 모든 과정을 시민에게 공개
```

**이 6가지가 유기적으로 연결되어, 세계 최초의 AI 운영 정당이 작동합니다.**
